# Scorching Ray
- **Level**: 2
- **School**: Evocation
- **Ritual**: false
- **Classes**: Sorcerer, Wizard

- **Casting Time Value**: 1
- **Casting Time Unit**: action
- **Combat Cost**: action

- **Range Type**: ranged
- **Range Distance**: 120
- **Targeting Type**: multi
- **Targeting Max**: 3
- **Valid Targets**: creatures
- **Line of Sight**: true

- **Verbal**: true
- **Somatic**: true
- **Material**: false

- **Duration Type**: instantaneous
- **Concentration**: false

- **Effect Type**: DAMAGE
- **Attack Roll**: ranged
- **Damage Dice**: 2d6
- **Damage Type**: Fire

- **Description**: You create three rays of fire and hurl them at targets within range. You can hurl them at one target or several. Make a ranged spell attack for each ray. On a hit, the target takes 2d6 Fire damage.
- **Higher Levels**: When you cast this spell using a spell slot of 3rd level or higher, you create one additional ray for each slot level above 2nd.
- **Source**: PHB 2024 p.315
- **Status**: Complete
