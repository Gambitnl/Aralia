# Invisibility
- **Level**: 2
- **School**: Illusion
- **Ritual**: false
- **Classes**: Bard, Sorcerer, Warlock, Wizard

- **Casting Time Value**: 1
- **Casting Time Unit**: action
- **Combat Cost**: action

- **Range Type**: touch
- **Targeting Type**: single
- **Valid Targets**: creature
- **Line of Sight**: true

- **Verbal**: true
- **Somatic**: true
- **Material**: true
- **Material Description**: an eyelash encased in gum arabic
- **Material Cost GP**: 0
- **Consumed**: false

- **Duration Type**: timed
- **Duration Value**: 1
- **Duration Unit**: hour
- **Concentration**: true

- **Effect Type**: BUFF
- **Save Stat**: None
- **Save Outcome**: none
- **Conditions Applied**: Invisible

- **Description**: A creature you touch becomes Invisible until the spell ends. Anything the target is wearing or carrying is Invisible as long as it is on the target's person. The spell ends for a target that attacks, deals damage, or casts a spell.
- **Higher Levels**: When you cast this spell using a spell slot of 3rd level or higher, you can target one additional creature for each slot level above 2nd.
- **Source**: PHB 2024 p.289
- **Status**: Complete
