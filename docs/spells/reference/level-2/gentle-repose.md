- **Level**: 2
- **School**: Necromancy
- **Ritual**: true
- **Classes**: Cleric, Paladin, Wizard
- **Sub-Classes**: Cleric - Grave Domain (XGtE), Druid - Circle of Spores (TCoE)

- **Targeting Type**: single
- **Valid Targets**: corpse
- **Line of Sight**: true

- **Verbal**: true
- **Somatic**: true
- **Material**: true
- **Material Description**: 2 Copper Pieces, which the spell consumes
- **Material Cost GP**: 0
- **Consumed**: true

- **Duration Type**: timed
- **Duration Value**: 10
- **Duration Unit**: day
- **Concentration**: false

- **Effect Type**: UTILITY
- **Utility Type**: other
- **Save Stat**: None
- **Save Outcome**: none

- **Description**: You touch a corpse or other remains. For the duration, the target is protected from decay and can't become Undead.
