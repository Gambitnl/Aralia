# Gentle Repose
- **Level**: 2
- **School**: Necromancy
- **Ritual**: true
- **Classes**: Cleric, Wizard

- **Casting Time Value**: 1
- **Casting Time Unit**: action
- **Combat Cost**: action

- **Range Type**: touch
- **Targeting Type**: single
- **Valid Targets**: corpse
- **Line of Sight**: true

- **Verbal**: true
- **Somatic**: true
- **Material**: true
- **Material Description**: a pinch of salt and one copper piece placed on each of the corpse's eyes
- **Material Cost GP**: 0
- **Consumed**: true

- **Duration Type**: timed
- **Duration Value**: 10
- **Duration Unit**: day
- **Concentration**: false

- **Effect Type**: UTILITY
- **Save Stat**: None
- **Save Outcome**: none

- **Description**: You touch a corpse or other remains. For the duration, the target is protected from decay and can't become Undead. The spell also effectively extends the time limit on raising the target from the dead, since days spent under the influence of this spell don't count against the time limit of spells such as Raise Dead.
- **Higher Levels**: None
- **Source**: PHB 2024 p.278
- **Status**: Complete
