# Remove Curse
- **Level**: 3
- **School**: Abjuration
- **Ritual**: false
- **Classes**: Cleric, Paladin, Warlock, Wizard

- **Casting Time Value**: 1
- **Casting Time Unit**: action
- **Combat Cost**: action

- **Range Type**: touch
- **Targeting Type**: single
- **Valid Targets**: creature_or_object
- **Line of Sight**: true

- **Verbal**: true
- **Somatic**: true
- **Material**: false

- **Duration Type**: instantaneous
- **Concentration**: false

- **Effect Type**: UTILITY
- **Save Stat**: None
- **Save Outcome**: none

- **Description**: At your touch, all curses affecting one creature or object end. If the object is a Cursed Magic Item, its curse remains, but the spell breaks its owner's Attunement to the object so it can be removed or discarded.
- **Higher Levels**: None
- **Source**: PHB 2024 p.314
- **Status**: Complete
