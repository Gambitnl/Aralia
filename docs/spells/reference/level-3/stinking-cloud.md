# Stinking Cloud
- **Level**: 3
- **School**: Conjuration
- **Ritual**: false
- **Classes**: Bard, Sorcerer, Wizard
- **Sub-Classes**: Warlock - Fiend Patron, Cleric - Festus Domain, Warlock - Mother of Sorrows (BoET)

- **Casting Time Value**: 1
- **Casting Time Unit**: action
- **Combat Cost**: action

- **Range Type**: ranged
- **Range Distance**: 90
- **Targeting Type**: area
- **Area Shape**: sphere
- **Area Size**: 20
- **Valid Targets**: creatures
- **Line of Sight**: true

- **Verbal**: true
- **Somatic**: true
- **Material**: true
- **Material Description**: a rotten egg
- **Material Cost GP**: 0
- **Consumed**: false

- **Duration Type**: timed
- **Duration Value**: 1
- **Duration Unit**: minute
- **Concentration**: true

- **Effect Type**: DEBUFF
- **Save Stat**: Constitution
- **Save Outcome**: negates
- **Conditions Applied**: Poisoned

- **Description**: You create a 20-foot-radius Sphere of yellow, nauseating gas centered on a point within range. The cloud is Heavily Obscured . The cloud lingers in the air for the duration or until a strong wind (such as the one created by Gust of Wind) disperses it. Each creature that starts its turn in the Sphere must succeed on a Constitution saving throw or have the Poisoned condition until the end of the current turn. While Poisoned in this way, the creature can't take an action or a Bonus Action.
- **Higher Levels**: None
