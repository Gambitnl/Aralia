# Creation
- **Level**: 5
- **School**: Illusion
- **Ritual**: false
- **Classes**: Sorcerer, Wizard

- **Casting Time Value**: 1
- **Casting Time Unit**: minute
- **Combat Cost**: long_cast

- **Range Type**: ranged
- **Range Distance**: 30
- **Targeting Type**: point
- **Valid Targets**: point
- **Line of Sight**: true

- **Verbal**: true
- **Somatic**: true
- **Material**: true
- **Material Description**: a paintbrush
- **Material Cost GP**: 0
- **Consumed**: false

- **Duration Type**: special
- **Concentration**: false

- **Effect Type**: UTILITY
- **Description**: You pull wisps of shadow material from the Shadowfell to create an object within range. It is either an object of vegetable matter (soft goods, rope, wood, and the like) or mineral matter (stone, crystal, metal, and the like). The object must be no larger than a 5-foot Cube, and the object must be of a form and material that you have seen. The spell's duration depends on the object's material. If the object is composed of multiple materials, use the shortest duration. Using any object created by this spell as another spell's Material component causes the other spell to fail.

  **Materials and Duration:**
  * Vegetable matter: 24 hours
  * Stone or crystal: 12 hours
  * Precious metals: 1 hour
  * Gems: 10 minutes
  * Adamantine or mithral: 1 minute

- **Higher Levels**: When you cast this spell using a spell slot of 6th level or higher, the Cube increases by 5 feet for each spell slot level above 5th.
- **Source**: PHB 2024
- **Status**: Complete
