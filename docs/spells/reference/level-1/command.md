# Command
- **Level**: 1
- **School**: Enchantment
- **Ritual**: false
- **Classes**: Bard, Cleric, Paladin

- **Casting Time Value**: 1
- **Casting Time Unit**: action
- **Combat Cost**: action

- **Range Type**: ranged
- **Range Distance**: 60
- **Targeting Type**: single
- **Valid Targets**: creature
- **Line of Sight**: true

- **Verbal**: true
- **Somatic**: false
- **Material**: false

- **Duration Type**: instantaneous
- **Concentration**: false

- **Effect Type**: STATUS
- **Save Stat**: Wisdom
- **Save Outcome**: negates
- **Conditions Applied**: Prone (Grovel)

- **Description**: You speak a one-word command to a creature you can see within range. The target must succeed on a Wisdom saving throw or follow the command on its next turn. The spell has no effect if the target is Undead, if it doesn't understand your language, or if your command is directly harmful to it.
- **Higher Levels**: When you cast this spell using a spell slot of 2nd level or higher, you can affect one additional creature for each slot level above 1st. The creatures must be within 30 feet of each other when you target them.
- **Source**: PHB 2024 p.251
- **Status**: Complete
