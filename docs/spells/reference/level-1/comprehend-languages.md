# Comprehend Languages
- **Level**: 1
- **School**: Divination
- **Ritual**: true
- **Classes**: Bard, Sorcerer, Warlock, Wizard

- **Casting Time Value**: 1
- **Casting Time Unit**: action
- **Combat Cost**: action

- **Range Type**: self
- **Targeting Type**: self
- **Valid Targets**: self
- **Line of Sight**: false

- **Verbal**: true
- **Somatic**: true
- **Material**: true
- **Material Description**: a pinch of soot and salt
- **Material Cost GP**: 0
- **Consumed**: false

- **Duration Type**: timed
- **Duration Value**: 1
- **Duration Unit**: hour
- **Concentration**: false

- **Effect Type**: UTILITY
- **Save Stat**: None
- **Save Outcome**: none

- **Description**: For the duration, you understand the literal meaning of any spoken language that you hear. You also understand any written language that you see, but you must be touching the surface on which the words are written. It takes about 1 minute to read one page of text. This spell doesn't decode secret messages in a text or a glyph, such as an arcane sigil, that isn't part of a written language.
- **Higher Levels**: None
- **Source**: PHB 2024 p.252
- **Status**: Complete
