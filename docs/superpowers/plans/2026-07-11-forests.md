# Forests Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Forests become named places with character (ordinary/ancient/haunted/fey), readable tree-fill rendering on the 2D map, POI destinations for the village forest paths, and a 3D interior that clusters into thickets, grows undergrowth, and dims under dense canopy.

**Spec:** `docs/superpowers/specs/2026-07-11-forests-design.md` — read it first.

**Architecture:** A new atlas-time forests pass (own seeded RNG streams, purely additive — the world-preservation doctrine from the roads campaign is binding) clusters contiguous forest cells into named `pack.forests`, assigns kinds, places forest POI markers, and retargets the existing village spur paths onto them. Consumers: a `'forest'` label kind in the existing label pipeline, a shared 2D tree-glyph module lighting up the dead FMG icon data in both renderers, an escalate-only biome override in `biomeForCell`, and a 3D layer (shared clearing noise + undergrowth + canopy light/fog modulation).

**Tech Stack:** TypeScript, Vitest, React (SVG atlas) + canvas, three.js/R3F. No new dependencies.

## Global Constraints

- **NO manual git commits, NO branches/worktrees.** Auto daily snapshot repo; where the template says "Commit", run the task's tests + `npx tsc --noEmit` instead.
- **Agora locks:** the controller (claude-roads) locks files before dispatch; implementers run NO Agora commands and touch ONLY their allowed files.
- **World-preservation doctrine (BINDING):** no new code may call `Math.random`, `rw()`, `gauss()`, `P()`, any `NamesGenerator` method, or any other shared-stream helper during world generation. New passes use `rngFromPath(streamPath(...))`-style own streams (`SeededRandom`; `nextInt` is MAX-EXCLUSIVE) or pure hashes. All new pack fields are additive; every pre-existing golden stays byte-identical. The fmgWorld golden suite is the enforcement net.
- Forest biome indices: `{5, 6, 7, 8, 9}` (tropical seasonal, temperate deciduous, tropical rainforest, temperate rainforest, taiga). Deep forest = `{7, 8, 9}` (matches roadTunables).
- Escalate-only biome override: haunted/fey forests override their cells to `forest_haunted`/`forest_fey`; ALL other cells keep today's mapping (including index 8 → `forest_ancient`).
- Tests per-file via `npx vitest run <path>`; `npx tsc --noEmit` must stay at the pre-existing error baseline (grep-compare; baseline ~520, none in touched files).
- US-English plain-language comments matching each file's density.
- ~346 empty test files + dev_hub.html build failure are known background noise.
