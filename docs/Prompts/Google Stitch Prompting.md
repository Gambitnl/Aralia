https://discuss.ai.google.dev/t/stitch-prompt-guide/83844

### Planner Response

A summary of the Stitch prompt guide was captured here, including:

- start broad or specific depending on the goal
- use a few clear adjectives to set the visual tone
- refine one screen and one or two changes at a time
- be specific about colors, fonts, imagery, and UI elements
- avoid mixing too many unrelated changes in one prompt
- save screenshots after each working step
