A premium, glassmorphic Character Equipment UI for a dark-themed D&D web application.
The UI should be built in React with Tailwind CSS.

Layout and Features:
1. The left side should feature an animated, glass-paneled "Paper Doll" or "Equipment Slots" view (Head, Chest, Main Hand, Off Hand, Ring, Wondrous Item, etc.).
2. The right side should feature an "Inventory Grid" showing unequipped items with high-quality icons and rarity color coding (Common, Uncommon, Rare, Very Rare, Legendary).
3. Clicking an item should open a beautifully styled "Item Stat Block" popover or side-panel showing:
   - Item Name and Type (e.g., "Longsword", "Martial Weapon")
   - Damage (e.g., "1d8 Slashing")
   - Properties (e.g., "Versatile (1d10)")
   - A rich, readable description.
   - Attunement requirement tag.

Design Aesthetics (CRITICAL):
- Use a stunning dark mode palette (deep slate backgrounds, vibrant accents).
- Implement Glassmorphism for panels: semi-transparent backgrounds with backdrop blur, subtle white/colored borders, and soft glows.
- Use curated, harmonious typography (e.g., Inter or Outfit font families).
- Add micro-animations: smooth hover states, soft glowing transitions when selecting items, and scale effects when hovering over inventory slots.
- Do NOT use generic red/blue/green. Use sleek, tailored HSL color tokens.
- Make it feel extremely premium, like a next-generation RPG interface.
