## 2024-05-23 - Weather System Foundations **Learning:** The codebase heavily utilizes `src/types/index.ts` as a central aggregation point, but domain-specific types should originate in their own files (e.g., `src/types/weather.ts`) to avoid monolithic files. **Action:** Future environmental systems (hazards, seasons) should follow this pattern of creating a dedicated type definition file and then re-exporting via index.
