Primary Commands

claude init [project-name]
Initializes a new Claude workspace in the current directory with default configuration.
claude sync --deep
Performs a deep sync of local context with the remote repository.
claude analyze ./src
Runs a static analysis pass over the specified directory and outputs recommendations.

Optional Parameters

Modify command execution behavior with these flags.

--force, -f
Bypass confirmation prompts and forcefully execute the action.
--verbose, -v
Enable detailed logging output for debugging purposes.
--timeout=<ms>
Specify the maximum execution time in milliseconds before aborting.

Slash Commands

Use these shortcuts while inside the interactive TUI mode.

/goal <description>
Sets a long-running autonomous objective for the agent.
/context clear
Wipes the current session memory to start a fresh conversation tree.
/export <format>
Dumps the current session state to a file (JSON or Markdown).
